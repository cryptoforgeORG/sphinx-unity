# Sphinx Unity Utility/Lightning Network Framework

Use this Unity3D tool to interface with your Sphinx relay and Sphinx.Chat.

### Features

- [x] Parse Restore Key
- [x] Get Contacts
- [x] Get Tribes
- [x] Get Messages

- [X] Send Tribe Message

- [ ] Send Payment
